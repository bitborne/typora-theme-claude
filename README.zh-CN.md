## 特色

- 🎨 **灵感来源于 claude.ai** — 整体视觉语言、配色和排版节奏均参考自 Claude 官网界面，暗色主题的背景色直接取色自 claude.ai。需要注意的是，本主题并非官网的精确复刻，部分设计有意做了差异化处理。
- 🌑 **暗色模式覆盖全面** — 暗色主题的样式覆盖远超一般主题，包括 HTML 块及 Mermaid 三种图表类型（流程图、时序图、甘特图），几乎不会出现视觉违和的地方。
- 🧪 **附带详尽的样式测试文档** — 内附完整的测试文件，覆盖 Typora 几乎所有样式元素：标题、列表、表格、引用、代码块、数学公式、图表、HTML 嵌入等，所见即所得。

## 安装

1. 从 [Releases](../../releases) 页面下载 `claude.css` 和 `claude-dark.css`。
2. 打开 Typora，进入 **偏好设置 → 外观 → 打开主题文件夹**。
3. 将两个 `.css` 文件复制到主题文件夹中。
4. 重启 Typora。
5. 进入 **偏好设置 → 外观 → 主题**，选择 **Claude**（亮色）或 **Claude Dark**（暗色）。
